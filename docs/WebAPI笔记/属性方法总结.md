### 方法

* concat()

  链接两个或者更多的数组

* join()

  把数组的所有元素都放入一个字符串.元素通过制定的分隔符进行分割

#### toString("")

​	可把数组转换为字符串，并返回结果。 使用传入数据分割这串字符串

* pop()
* 删除并返回数组的最后一个元素
  * 数组长度-1
  * 该方法会直接修改原有数组

* push()
  * 向数组的末尾添加一个或更多元素，并返回新的长度。(可以添加多个元素,返回值为新的长度)
  * 该方法会直接修改原有数组
* shift()

删除并返回数组的第一个元素

该方法会直接修改原有数组

* slice(start,end)----可从已有的数组中返回选定的元素。

* start
  * 必需。规定从何处开始选取。如果是负数，那么它规定从数组尾部开始算起的位置。也就是说，-1 指最后一个元素，-2 指倒数第二个元素，以此类推。
* end
  * 可选。规定从何处结束选取。该参数是数组片断结束处的数组下标。如果没有指定该参数，那么切分的数组包含从 start 到数组结束的所有元素。如果这个参数是负数，那么它规定的是从数组尾部开始算起的元素。
* 返回值
  * 返回一个新的数组，包含从 start 到 end （不包括该元素）的 arrayObject 中的元素。

### 定时器

* 说明,上面两个方法都是window对象下的方法；执行定时器，都是等待自己的间隔才开始执行；

#### setTimeout

* 一次性定时器

```js
var timer = setTimeout(函数,延迟的毫秒数);

// 停止一次性的定时器：清除后，就不会执行这个定时器；

clearTimeout(timer);
```

#### setInterval

* 永久性定时器
  * 需要清除定时器的时候需要把定时器的返回值赋值给变量, 清除的时候吧变量传入就可以清除定时器

* 作用:
  * 阶段的时间执行函数；
* 返回值
  * 就是该定时器的id

```js
var timer = setInterval(函数,间隔毫秒数);

// 清除永久定时器

clearInterval(timer);
```

